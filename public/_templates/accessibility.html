Accessibility Statement

This site is made for people. As such I try my best to be at least AA compliant with the Web Content Accessibility Guidelines (WCAG) and make the web more accessible and inclusive.

That said, I'm not perfect and I'm always learning and improving the website.

I run automated testing and manual testing for accessibility issues regularly using the following tools:

  • WAVE for automated testing
  • Edge on Windows 11 and NVDA
  • Edge on Android and TalkBack
  • Edge on Windows 11 and Android with 200% zoom

If you have any issues with the site, please feel free to contact me at me@madcampos.dev.

If you have the technical knowledge and would like to, feel free to open an issue on GitHub.
